Selection Sort Visualisierung

Funktionsweise: Selection Sort durchläuft das Array und findet bei jedem Durchlauf das kleinste (oder größte) Element im unsortierten Teil. Dieses Element wird dann mit dem ersten Element des unsortierten Teils vertauscht. Der sortierte Teil wächst so von links nach rechts.

Implementierung in Python

# Selection Sort Implementierung in Python def selection_sort(arr): """ Sortiert ein Array mit dem Selection Sort Algorithmus. Zeitkomplexität: O(n²) Platzkomplexität: O(1) """ n = len(arr) # Durchlaufe das Array for i in range(n - 1): # Finde das Minimum im unsortierten Teil min_idx = i for j in range(i + 1, n): if arr[j] < arr[min_idx]: min_idx = j # Tausche das gefundene Minimum mit dem ersten Element if min_idx != i: arr[i], arr[min_idx] = arr[min_idx], arr[i] return arr # Verwendungsbeispiel array = [64, 34, 25, 12, 22, 11, 90] print("Unsortiert:", array) selection_sort(array) print("Sortiert:", array)

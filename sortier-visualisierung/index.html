Sortieralgorithmen Visualisierung

Interaktive Visualisierung verschiedener Sortieralgorithmen

Bubble Sort

Vergleicht benachbarte Elemente und tauscht sie, wenn sie in falscher Reihenfolge sind.

Zeitkomplexität: O(n²)
Platzkomplexität: O(1)

Quick Sort

Teile-und-herrsche-Algorithmus, der ein Pivot-Element verwendet.

Zeitkomplexität: O(n log n)
Platzkomplexität: O(log n)

Merge Sort

Teilt das Array rekursiv und fügt die sortierten Teile zusammen.

Zeitkomplexität: O(n log n)
Platzkomplexität: O(n)

Insertion Sort

Baut eine sortierte Liste auf, indem Elemente an der richtigen Position eingefügt werden.

Zeitkomplexität: O(n²)
Platzkomplexität: O(1)

Selection Sort

Findet wiederholt das kleinste Element und platziert es an der richtigen Position.

Zeitkomplexität: O(n²)
Platzkomplexität: O(1)

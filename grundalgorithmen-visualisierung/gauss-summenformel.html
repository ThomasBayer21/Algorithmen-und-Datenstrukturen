Gaußsche Summenformel

Überblick: Die Gaußsche Summenformel ist eine elegante mathematische Formel, die die Summe aller natürlichen Zahlen von 1 bis n in konstanter Zeit O(1) berechnet, anstatt iterativ durch alle Zahlen zu gehen O(n). Die berühmte Geschichte erzählt, dass der junge Carl Friedrich Gauß diese Formel in der Grundschule entdeckte.

Die Formel

1 + 2 + 3 + ... + n = n × (n + 1) / 2

Wie funktioniert es?

Die Grundidee ist, dass man die Zahlen paarweise von außen nach innen addiert:

  • 1 + n = n + 1
  • 2 + (n-1) = n + 1
  • 3 + (n-2) = n + 1
  • ...

Es gibt genau n/2 solcher Paare, jedes mit der Summe (n+1).

Also: Summe = n/2 × (n+1) = n × (n+1) / 2

Code-Vergleich

Naive iterative Methode - O(n)

function sumIterative(n) { let sum = 0; for (let i = 1; i <= n; i++) { sum += i; // n Iterationen } return sum; } // Zeitkomplexität: O(n) // Raumkomplexität: O(1)

Gaußsche Formel - O(1)

function sumGauss(n) { return n * (n + 1) / 2; // Nur eine Berechnung! } // Zeitkomplexität: O(1) - konstante Zeit! // Raumkomplexität: O(1)

Praktische Anwendungen

Wo wird die Gaußsche Summenformel verwendet?

  • Algorithmische Optimierung: Vermeidung unnötiger Schleifen
  • Mathematische Beweise: Summen von arithmetischen Reihen
  • Speicherzugriffsmuster: Berechnung von Array-Indizes in mehrdimensionalen Arrays
  • Kombinatorik: Anzahl der Kanten in vollständigen Graphen (n Knoten haben n(n-1)/2 Kanten)
  • Zeitkomplexität: Analyse verschachtelter Schleifen

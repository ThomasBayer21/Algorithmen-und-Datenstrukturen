Funktionsweise: Die binäre Suche funktioniert nur auf sortierten Arrays. Sie teilt das Suchintervall wiederholt in der Mitte und vergleicht das mittlere Element mit dem gesuchten Wert. Abhängig vom Vergleich wird dann in der linken oder rechten Hälfte weitergesucht. Dies führt zu einer logarithmischen Zeitkomplexität O(log n).

Implementierung in Python

# Binäre Suche Implementierung in Python def binary_search(arr, target): """ Sucht ein Element in einem sortierten Array mit binärer Suche. Zeitkomplexität: O(log n) Platzkomplexität: O(1) Args: arr: Sortiertes Array target: Zu suchendes Element Returns: Index des Elements oder -1 falls nicht gefunden """ left = 0 right = len(arr) - 1 while left <= right: # Berechne mittleren Index mid = (left + right) // 2 # Element gefunden if arr[mid] == target: return mid # Target ist größer, suche in rechter Hälfte elif arr[mid] < target: left = mid + 1 # Target ist kleiner, suche in linker Hälfte else: right = mid - 1 # Element nicht gefunden return -1 # Verwendungsbeispiel array = [11, 15, 22, 28, 34, 41, 50, 57, 64, 72] target = 50 result = binary_search(array, target) if result != -1: print(f"Element {target} gefunden an Position {result}") else: print(f"Element {target} nicht gefunden")

Suchalgorithmen Visualisierung

Interaktive Visualisierung verschiedener Suchalgorithmen

Lineare Suche

Durchsucht das Array sequenziell von links nach rechts, bis das gesuchte Element gefunden wird.

Zeitkomplexität: O(n)
Platzkomplexität: O(1)
Bedingung: Keine

Binäre Suche

Teilt das Array wiederholt in der Mitte und sucht in der entsprechenden Hälfte weiter.

Zeitkomplexität: O(log n)
Platzkomplexität: O(1)
Bedingung: Sortiertes Array

Suchalgorithmen Visualisierung

Interaktive Visualisierung verschiedener Suchalgorithmen

Lineare Suche

Durchsucht das Array sequenziell von links nach rechts, bis das gesuchte Element gefunden wird.

Zeitkomplexität: O(n)
Platzkomplexität: O(1)
Bedingung: Keine

Binäre Suche

Teilt das Array wiederholt in der Mitte und sucht in der entsprechenden Hälfte weiter.

Zeitkomplexität: O(log n)
Platzkomplexität: O(1)
Bedingung: Sortiertes Array

Tiefensuche (DFS)

Traversiert einen Graphen, indem zuerst in die Tiefe gegangen wird, bevor zurückgegangen wird (Backtracking).

Zeitkomplexität: O(V + E)
Platzkomplexität: O(V)
Verwendung: Stack/Rekursion

Breitensuche (BFS)

Traversiert einen Graphen ebenenweise, besucht alle Knoten einer Ebene vor der nächsten Ebene.

Zeitkomplexität: O(V + E)
Platzkomplexität: O(V)
Verwendung: Queue

Backtracking

Überblick: Backtracking ist eine algorithmische Technik zur systematischen Suche nach Lösungen. Es probiert mögliche Lösungen aus und macht einen Schritt zurück (backtrack), wenn eine Sackgasse erreicht wird. Typische Anwendungen sind das N-Damen-Problem, Sudoku-Solver und Labyrinth-Lösungen.

1. Labyrinth lösen

Algorithmus:

  1. Starte an der Startposition
  2. Versuche alle vier Richtungen (oben, rechts, unten, links)
  3. Wenn eine Richtung gültig ist, gehe einen Schritt weiter
  4. Wenn Ziel erreicht: Fertig!
  5. Wenn Sackgasse: Gehe zurück (Backtrack) und versuche andere Richtung
function solveMaze(maze, x, y, path) { // Basisfall: Ziel erreicht if (x === endX && y === endY) { return true; } // Prüfe ob Position gültig ist if (!isValid(x, y)) { return false; } // Markiere als besucht visited[x][y] = true; path.push({x, y}); // Versuche alle 4 Richtungen const directions = [[0,1], [1,0], [0,-1], [-1,0]]; for (const [dx, dy] of directions) { if (solveMaze(maze, x+dx, y+dy, path)) { return true; // Lösung gefunden! } } // Backtrack: Dieser Weg führt nicht zur Lösung path.pop(); return false; }

2. N-Damen-Problem

Problem:

Platziere N Damen auf einem N×N Schachbrett, so dass keine Dame eine andere angreifen kann. Eine Dame kann horizontal, vertikal und diagonal angreifen.

function solveNQueens(row, board, n) { // Alle Damen platziert - Lösung gefunden! if (row === n) { return true; } // Versuche Dame in jeder Spalte zu platzieren for (let col = 0; col < n; col++) { if (isSafe(board, row, col, n)) { // Platziere Dame board[row][col] = 1; // Rekursiv nächste Zeile lösen if (solveNQueens(row + 1, board, n)) { return true; } // Backtrack: Dame entfernen board[row][col] = 0; } } return false; // Keine Lösung in diesem Pfad }

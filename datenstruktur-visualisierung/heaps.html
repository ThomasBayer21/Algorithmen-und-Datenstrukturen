Heaps (Priority Queue)

Überblick: Ein Heap ist eine baumartige Datenstruktur, die als Array implementiert wird. In einem Max-Heap ist jeder Elternknoten größer als seine Kinder, in einem Min-Heap ist jeder Elternknoten kleiner. Heaps werden für Priority Queues und Heap Sort verwendet.

1. Max-Heap

Eigenschaften:

Heap-Eigenschaft: Parent ≥ Children

Einfügen: O(log n)

Extract-Max: O(log n)

Get-Max: O(1)

Array-Implementierung: Parent at index i, Children at 2i+1 und 2i+2

// Max-Heap Klasse class MaxHeap { constructor() { this.heap = []; } // Einfügen - O(log n) insert(value) { this.heap.push(value); this.heapifyUp(this.heap.length - 1); } // Nach oben "bubblen" heapifyUp(index) { let parentIndex = Math.floor((index - 1) / 2); if (index > 0 && this.heap[index] > this.heap[parentIndex]) { [this.heap[index], this.heap[parentIndex]] = [this.heap[parentIndex], this.heap[index]]; this.heapifyUp(parentIndex); } } // Maximum entfernen - O(log n) extractMax() { if (this.heap.length === 0) return null; const max = this.heap[0]; this.heap[0] = this.heap.pop(); if (this.heap.length > 0) { this.heapifyDown(0); } return max; } // Nach unten "bubblen" heapifyDown(index) { let largest = index; const left = 2 * index + 1; const right = 2 * index + 2; if (left < this.heap.length && this.heap[left] > this.heap[largest]) { largest = left; } if (right < this.heap.length && this.heap[right] > this.heap[largest]) { largest = right; } if (largest !== index) { [this.heap[index], this.heap[largest]] = [this.heap[largest], this.heap[index]]; this.heapifyDown(largest); } } }

2. Heap Sort

Algorithmus:

  1. Baue einen Max-Heap aus dem Array
  2. Tausche Root (Maximum) mit letztem Element
  3. Reduziere Heap-Größe um 1
  4. Heapify Root
  5. Wiederhole 2-4 bis Heap leer ist

Komplexität: O(n log n) in allen Fällen

// Heap Sort - O(n log n) function heapSort(arr) { const n = arr.length; // 1. Build max heap for (let i = Math.floor(n / 2) - 1; i >= 0; i--) { heapify(arr, n, i); } // 2. Extract elements one by one for (let i = n - 1; i > 0; i--) { [arr[0], arr[i]] = [arr[i], arr[0]]; heapify(arr, i, 0); } return arr; }

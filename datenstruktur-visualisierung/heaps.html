Heaps (Priority Queue)

Überblick: Ein Heap ist eine baumartige Datenstruktur, die als Array implementiert wird. In einem Max-Heap ist jeder Elternknoten größer als seine Kinder, in einem Min-Heap ist jeder Elternknoten kleiner. Heaps werden für Priority Queues und Heap Sort verwendet.

1. Max-Heap

Eigenschaften:

Heap-Eigenschaft: Parent ≥ Children

Einfügen: O(log n)

Extract-Max: O(log n)

Get-Max: O(1)

Array-Implementierung: Parent at index i, Children at 2i+1 und 2i+2

# Max-Heap Klasse class MaxHeap: def __init__(self): self.heap = [] # Einfügen - O(log n) def insert(self, value): self.heap.append(value) self._heapify_up(len(self.heap) - 1) # Nach oben "bubblen" def _heapify_up(self, index): parent_index = (index - 1) // 2 if index > 0 and self.heap[index] > self.heap[parent_index]: self.heap[index], self.heap[parent_index] = \ self.heap[parent_index], self.heap[index] self._heapify_up(parent_index) # Maximum entfernen - O(log n) def extract_max(self): if len(self.heap) == 0: return None max_val = self.heap[0] self.heap[0] = self.heap.pop() if len(self.heap) > 0: self._heapify_down(0) return max_val # Nach unten "bubblen" def _heapify_down(self, index): largest = index left = 2 * index + 1 right = 2 * index + 2 if left < len(self.heap) and self.heap[left] > self.heap[largest]: largest = left if right < len(self.heap) and self.heap[right] > self.heap[largest]: largest = right if largest != index: self.heap[index], self.heap[largest] = \ self.heap[largest], self.heap[index] self._heapify_down(largest)

2. Heap Sort

Algorithmus:

  1. Baue einen Max-Heap aus dem Array
  2. Tausche Root (Maximum) mit letztem Element
  3. Reduziere Heap-Größe um 1
  4. Heapify Root
  5. Wiederhole 2-4 bis Heap leer ist

Komplexität: O(n log n) in allen Fällen

# Heap Sort - O(n log n) def heap_sort(arr): n = len(arr) # 1. Build max heap for i in range(n // 2 - 1, -1, -1): heapify(arr, n, i) # 2. Extract elements one by one for i in range(n - 1, 0, -1): arr[0], arr[i] = arr[i], arr[0] heapify(arr, i, 0) return arr

Arrays und Felder

Überblick: Arrays sind die einfachste und fundamentalste Datenstruktur. Sie speichern Elemente in zusammenhängenden Speicherbereichen und ermöglichen konstanten Zugriff per Index. Diese Seite zeigt verschiedene Array-Operationen und ihre Komplexität.

1. Eindimensionales Array

Eigenschaften:

Zugriff: O(1) - Direkter Zugriff über Index

Suche: O(n) - Lineares Durchsuchen erforderlich

Einfügen/Löschen: O(n) - Elemente müssen verschoben werden

# Array-Zugriff - O(1) def access_element(arr, index): return arr[index] # Konstante Zeit # Array-Suche - O(n) def search_element(arr, value): for i in range(len(arr)): if arr[i] == value: return i return -1 # Element einfügen - O(n) def insert_at(arr, index, value): # Alle Elemente ab index nach rechts verschieben arr.insert(index, value) return arr

2. Mehrdimensionale Arrays (Matrix)

2D Array (Matrix):

Ein Array von Arrays. Zugriff erfolgt mit zwei Indizes: matrix[row][col]

Speicherbedarf: O(n × m)

Zugriff: O(1)

Interaktive Matrix (3×3)

# 2D Array erstellen matrix = [ [1, 2, 3], [4, 5, 6], [7, 8, 9] ] # Zugriff auf Element element = matrix[1][2] # 6 # Matrix traversieren - O(n × m) def traverse_matrix(matrix): for row in matrix: for element in row: print(element)

3. Array-Operationen im Vergleich

Komplexitäten im Überblick:

Operation Zeitkomplexität Erklärung
Zugriff per Index O(1) Direkter Speicherzugriff
Suche (unsortiert) O(n) Muss alle Elemente durchsuchen
Einfügen (Anfang/Mitte) O(n) Elemente verschieben
Einfügen (Ende) O(1) Kein Verschieben nötig
Löschen (Anfang/Mitte) O(n) Elemente verschieben

Implementierung in Python

# Arrays in Python (Listen) # Array erstellen arr = [10, 25, 33, 47, 52] # Zugriff auf Element - O(1) def access_element(arr, index): """ Direkter Zugriff auf Element per Index Zeitkomplexität: O(1) """ return arr[index] # Element suchen - O(n) def search_element(arr, value): """ Lineare Suche nach einem Element Zeitkomplexität: O(n) """ for i in range(len(arr)): if arr[i] == value: return i return -1 # Element einfügen - O(n) def insert_at(arr, index, value): """ Element an bestimmter Position einfügen Zeitkomplexität: O(n) - Alle Elemente ab index müssen verschoben werden """ arr.insert(index, value) return arr # Element löschen - O(n) def delete_at(arr, index): """ Element an bestimmter Position löschen Zeitkomplexität: O(n) - Alle Elemente nach index müssen verschoben werden """ del arr[index] return arr # 2D Array (Matrix) erstellen und verwenden matrix = [ [1, 2, 3], [4, 5, 6], [7, 8, 9] ] # Matrix traversieren - O(n × m) def traverse_matrix(matrix): """ Alle Elemente einer Matrix durchlaufen Zeitkomplexität: O(n × m) wobei n = Zeilen, m = Spalten """ for row in matrix: for element in row: print(element, end=' ') print() # Verwendungsbeispiel arr = [10, 25, 33, 47, 52] print(f"Element an Index 2: {access_element(arr, 2)}") # 33 print(f"Index von 47: {search_element(arr, 47)}") # 3
